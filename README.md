Secured-Hash
es un programa que genera y chequea hashes de archivos o cadenas de
caracteres. &nbsp;Se ha desarrollado con Harbour Compiler 3.0
(compatible con Visual FoxPro 6.0). &nbsp;La velocidad de proceso
de
archivos actual es
lenta,
aproximadamente 30 minutos para un archivo de unos 350 kB. &nbsp;La
versión de Windows es la que ofrece mayor velocidad.
&nbsp;Esto se solucionará cuando pueda hacer una versión en otro
compilador más eficiente. &nbsp;Para
acceder a la ayuda del programa introduzca lo siguiente:<br>
      <br>
      <div style="text-align: center;">shash&nbsp; -h<br>
      </div>

Secured-Hash
is a file &amp; string hash generator &amp; checker.
&nbsp;It has been
created using Harbour 3.0 Compiler (Visual FoxPro 6.0 compatibility).
&nbsp;Current
speed for processing files is slow, approximately 30 min from a 350 kB
file. &nbsp;The Windows release gives you more speed.
&nbsp;This
problem will be solved when I can create the new
release using a more efficient compiler. &nbsp;To get help for this
program type next:<br>
      <br>
      <div style="text-align: center;">shash &nbsp;-h<br>
